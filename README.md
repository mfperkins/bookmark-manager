# Bookmark Manager

```
User stories

As a User
So that I can use the Bookmark Manager
I'd like to see a list of stored bookmarks links

As a User
So that I can find links later
I'd like to be able to add new links

As a User
So that I can categorise links
I'd like to be able to add tags to links

As a User
So that I can find specific links in a category
I'd like to be able to filter by tags
```
